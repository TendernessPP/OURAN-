# Vue2.0OURAN博客
<hr>

##安装：
1、安装node：http://nodejs.cn/download/
     git：https://git-scm.com/downloads

2、从我的仓库复制代码：
> $ git clone https://github.com/TendernessPP/OURAN-.git

3、安装vue脚手架工具vue-cli:
> $npm install vue-cli -g

4、进入代码根目录安装依赖：
> $ npm install --save-dev

5、运行命令：
> $ npm run dev

6、发布代码：
> $ npm run build
